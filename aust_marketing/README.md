# Aust Marketing Website

Este projeto é um site profissional e inovador para a agência de marketing **Aust**, focado no mercado LATAM.

## Estrutura do Projeto

- **index.html**: Estrutura semântica e conteúdo do site.
- **style.css**: Estilização completa com tema dark, animações e design responsivo.
- **script.js**: Interações avançadas, como cursor personalizado e efeitos de scroll reveal.

## Design System

- **Cores Principais**: Preto (`#050505`) e Branco (`#FFFFFF`).
- **Cor de Destaque**: Laranja Sofisticado (`#FF4D00`).
- **Tipografia**: 'Outfit' (Corpo) e 'Space Grotesk' (Títulos/Display).
- **Estilo**: Minimalista, Tecnológico, "Glassmorphism", Animações Fluidas.

## Como Visualizar

Basta abrir o arquivo `index.html` em qualquer navegador moderno.
